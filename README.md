# Hotel Terabithia 🏨

Sistema interno de operações hoteleiras desenvolvido para gerenciamento de reservas, hóspedes, eventos e serviços do hotel.

## 🌐 Deploy

Acesse o projeto online:

[https://hotel-terabithia-beryl.vercel.app/
](https://instragam-clone.vercel.app/)
---

# 📋 Sobre o Projeto

O Hotel Terabithia é uma plataforma administrativa criada para auxiliar funcionários do hotel em tarefas operacionais do dia a dia.

O sistema foi desenvolvido com foco em:

- Modularização
- Organização de código
- Interface moderna
- Experiência do usuário
- Validações e lógica de programação

---

#  Funcionalidades

##  Login
- Sistema de autenticação
- Controle de tentativas
- Mensagens personalizadas

##  Reservas de Quartos
- Escolha de quartos
- Tipos de acomodação:
  - Standard
  - Executivo
  - Luxo
- Cálculo automático:
  - subtotal
  - taxa de serviço
  - total final
- Confirmação de reserva
- Controle de quartos ocupados

##  Cadastro de Hóspedes
- Cadastro em memória
- Pesquisa de hóspedes
- Atualização e remoção
- Listagem organizada

## 🎉 Eventos
- Escolha automática de auditório
- Controle de disponibilidade
- Cálculo de garçons
- Buffet
- Relatório completo

##  Ar-Condicionado
- Comparação entre empresas
- Aplicação de descontos
- Melhor orçamento automático

##  Abastecimento
- Comparação entre postos
- Melhor combustível
- Melhor custo-benefício

##  Relatórios Operacionais
- Taxa de ocupação
- Receita acumulada
- Eventos confirmados
- Reservas realizadas
